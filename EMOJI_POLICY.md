# Emoji Policy - Subtle, Sophisticated, Data-Driven

**Philosophy**: Elegance over excitement. Data over decoration. If it's not adding strategic value, delete it.

---

## APPROVED (Minimal Use Only)
Use sparingly - never more than 2-3 per document:

### Status Indicators (Minimal)
- ✓ Checkmark (black, subtle confirmation)
- ✗ X mark (denial/error)
- ⚠ Warning (no color variation - plain yellow triangle)

### Data Visualization (ALWAYS with numbers)
- **NEVER use chart emojis alone** - always pair with actual metrics
- Example: "Pipeline velocity: 47 days (Q3 data)" - NO 📊
- Example: "$14.2M identified, 21 toxic accounts flagged" - NO 💰

**Rule**: If you're showing metrics, use actual numbers in text. No symbolic representations.

---

## BANNED (Everything Else)
If it's not on the approved list, it's banned. Period.

### Absolutely Forbidden
- 💰 💵 💳 Money bags/symbols (use "$14.2M" in text)
- 📊 📈 📉 Charts/graphs (show actual numbers instead)
- 📞 📧 📱 Communication icons (say "Call scheduled" instead)
- 🏆 Trophy (say "Won" or "Closed" instead)
- 🔑 Key (say "Critical" or "Strategic" instead)
- ✅ 🟢 🔴 🟡 Colored checkmarks/dots (use plain ✓ or ✗ only)
- 🎯 Target (say "Goal" or "Objective" instead)
- 📍 📌 ⭐ Pins/stars (just bold the text)
- 🎉 🎊 🥳 Celebrations (never)
- 😀 😃 😍 🤔 👍 All faces and hands (never)
- 💡 🔥 ⚡ Symbolic metaphors (use precise language)

### The Real Rule
**Use actual data, not symbols**:
- BAD: "📊 Pipeline performance is strong 📈"
- GOOD: "Pipeline velocity: 47 days (12% faster than Q2)"

- BAD: "🏆 Won Tesla account!"
- GOOD: "Tesla - Closed. $2.4M ACV, 18-month cycle."

- BAD: "🔑 Key stakeholders identified"
- GOOD: "Buying committee mapped: CFO, CIO, 3 VPs confirmed"

---

## REPLACEMENT GUIDE

Stop using visual shortcuts. Use precise business language:

| ❌ BANNED | ✓ APPROVED |
|-----------|------------|
| 📊 Performance up | "Q3 pipeline: $47M (18% QoQ growth)" |
| 💰 Revenue impact | "$14.2M incremental revenue identified" |
| 🏆 Big win | "Tesla closed - $2.4M ACV" |
| 🔥 Hot account | "Tier 1A priority - 90% fit score" |
| 🎯 Target | "Objective: 47-day velocity by Q4" |
| 📞 Call scheduled | "Discovery call: Feb 12, 2PM ET" |
| ✅ Done | "Completed" or just ✓ |
| 🚨 Urgent | "Critical: Response required by EOD" |

---

## ENFORCEMENT

**Test**: Would McKinsey put this in a slide?
- No → Delete it immediately

**Spacing Rule**: Elegance = white space + precision
- Spread data points with line breaks
- One metric per line
- No visual clutter

**Data Rule**: Every claim needs a number
- "Strong performance" → Banned
- "47-day velocity, 12% improvement" → Approved

---

## APPROVED SYMBOLS (Rare Use)

Only these, only when absolutely necessary:

```
✓ ✗ ⚠
```

That's it. Three symbols. Use sparingly.

---

**Bottom line**: If you're reaching for an emoji, write the data instead. Execs read numbers, not clipart.
